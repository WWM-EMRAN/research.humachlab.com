# HUMACH Logo Asset Update

## Purpose

The uploaded square and rectangular HUMACH Research logos were processed and integrated into the static website so that each format is used in the most appropriate location.

## Source Files Used

- `hmr_logo_sq.png` — square logo/icon artwork
- `hmr_logo_rc.png` — rectangular/horizontal HUMACH Research wordmark

## Generated Logo Assets

The processed files were saved in:

```text
assets/img/brand/
```

## Styling Updates

The stylesheet `assets/css/humach-theme.css` was updated so that:

- the square header logo remains compact and visible on the dark sticky header;
- the footer uses the horizontal logo inside a clean white rounded capsule for contrast;
- the logo layout remains responsive on mobile screens.
